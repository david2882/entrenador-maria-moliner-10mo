# Entrenador María Moliner 10mo

PWA para practicar el concurso de ortografía.

Incluye banco de 500 palabras, pregón con voz, práctica oral, simulacro de 5 rondas, banco de errores, tutor socrático y progreso local.

Para probarla: abre index.html o activa GitHub Pages desde la rama main.

El PDF oficial trae las palabras, pero no desarrolla todas las acepciones. Usa Editar ficha para completar definiciones verificadas.
